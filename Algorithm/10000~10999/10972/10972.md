> 다음 순열<p></p>
> 1부터 N까지의 수로 이루어진 순열이 주어진다.<br>
> 해당 순열을 사전순으로 정렬한다고 할 때, 다음번에 올 순열을 구해보자<br>
### 코드
```c++
#include<cstdio>
#include<algorithm>
using namespace std;
int n,arr[10002];
int main(){
    int loc=0;
    scanf("%d",&n);
    for(int i=0;i<n;i++){
        scanf("%d",&arr[i]);
        if(i && arr[i]>arr[i-1])
            loc=i;
    }
    if(!loc)
        printf("-1");
    else{
        next_permutation(arr,arr+n);
        for(int i=0;i<n;i++)
            printf("%d ",arr[i]);
    }
    return 0;
}
```
### 풀이
알고리즘 헤더의 next_permutation 함수를 이용해 문제를 해결했다.<p>
해당 헤더를 사용하지 않고 구현한 코드는 곧 추가할 예정<br>
