> 문자열 집합<p></p>
> n개의 문자열로 구성된 집합 S가 주어진다.<br>
> 이후, M개의 문자열을 입력받으며, S에 포함된 문자열의 수를 구하는 문제
### 코드
#### python
```py
n,m = map(int,input().split())
str_dict={}
sum_of_str=0
for i in range(n):
    str_in=input()
    str_dict[str_in]=1
for j in range(m):
    str_find=input()
    sum_of_str+=bool(str_dict.get(str_find))
print(sum_of_str)
```

### 풀이
#### 파이썬
파이썬의 딕셔너리 구조를 활용해 문제를 해결했다.<p></p>
N개의 문자열을 입력받으며, 빈 딕셔너리에 원소를 채워나간다.<br>
이후, M개의 문자열을 입력받고, get메소드를 통해 딕셔너리의 값을 가져와 이를 더해준다.
